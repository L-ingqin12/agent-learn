# 第一部分：AI Agent 概述

## 1.1 什么是 AI Agent

AI Agent（智能体）是一个能够**感知环境、做出决策、执行行动**的自主系统。它以大语言模型（LLM）为核心推理引擎，通过调用工具（Tools）、访问记忆（Memory）、执行规划（Planning）来完成复杂任务。

### Agent 与传统程序的区别

| 维度 | 传统程序 | AI Agent |
|------|---------|----------|
| 输入处理 | 结构化、确定性 | 自然语言、模糊性 |
| 决策逻辑 | 硬编码规则 | LLM 推理 |
| 能力边界 | 固定 | 可通过工具扩展 |
| 错误处理 | 预设路径 | 动态重试与反思 |

## 1.2 Agent 的核心能力

1. **感知（Perception）** — 理解用户意图和上下文
2. **推理（Reasoning）** — 分解任务、分析问题
3. **行动（Action）** — 调用工具执行具体操作
4. **记忆（Memory）** — 保存和使用历史信息
5. **学习（Learning）** — 从反馈中改进

## 1.3 Agent 架构模式

```
用户输入 → Agent 核心（LLM）→ 推理/规划 → 工具调用 → 结果返回
                ↑                ↓
            记忆系统 ←───── 观察与反馈
```

常见的架构模式：
- **ReAct（Reasoning + Acting）** — 交替进行推理和行动
- **Plan-and-Execute** — 先规划再逐步执行
- **Multi-Agent** — 多个 Agent 协作完成复杂任务
